# Review Vocabulary

Use one approved problem label for each review finding. If one span has multiple
independent problems, split it into separate findings. Do not include directives
or proposed fixes in Review mode.

## Problem Labels

| Label | Problem type |
|---|---|
| `[WORDY]` | Redundant words, throat-clearing, inflated phrasing |
| `[VAGUE]` | Imprecise quantifier, undefined term, unclear referent |
| `[PASSIVE]` | Passive voice where active would be clearer |
| `[TENSE]` | Wrong or inconsistent verb tense for the section |
| `[AMBIGUOUS]` | Pronoun, modifier, or referent with >1 possible meaning |
| `[OVERSTATED]` | Claim exceeds what the data support |
| `[HEDGED]` | Excessive or stacked hedging weakens the claim |
| `[JARGON]` | Undefined technical term or unexplained abbreviation |
| `[NOMINALIZED]` | Verb converted to noun+verb combination |
| `[INCONSISTENT]` | Term, unit, or notation differs from earlier usage |
| `[STAT-LANG]` | Incorrect statistical language, such as "highly significant" |
| `[ANTHROPOMORPHIC]` | Non-human entity given human intent or agency |
| `[MODIFIER]` | Dangling or misplaced modifier |
| `[PARALLEL]` | List or series breaks parallel structure |
| `[PRECISION]` | Number reported with wrong decimal places or missing units |
| `[CAUSAL]` | Causal claim from associational/observational data |
| `[PRONOUN-AGREE]` | Pronoun-antecedent mismatch |
| `[SUBJECT-VERB]` | Subject-verb agreement error |
| `[ABBREV]` | Abbreviation used before definition, or defined but used only once |
| `[FLOW]` | Paragraph or section lacks topic sentence, transition, or logical link |

## Directives

Use these directive phrases only when the user asks for revision, application,
or an action plan. Do not include them in review findings.

- → Shorten.
- → Quantify.
- → Clarify referent.
- → Use active voice.
- → Soften claim.
- → Strengthen claim.
- → Define at first use.
- → Check tense for this section.
- → Separate observation from interpretation.
- → Replace noun phrase with verb.
- → Align with earlier usage.
- → Add topic sentence.
- → Add transition.
- → Restructure as parallel items.
- → Specify units.
- → Remove or reattach modifier.
- → Correct statistical language.
- → Replace causal language with associational.

## Review Finding Shape

```text
[LABEL] [severity] "short quoted span"
Problem: one sentence explaining what is wrong and why.
```
